Code and Notes -

So you want to be a programer, but have no idea what that actually means. This class will go over the core concepts of programming in general to give students a better idea of how to write programs and be able to deploy solutions.

Programming requires an understanding of what languages are appropriate for which tasks. How to determine requirements for a programming project. How to design the look and flow of a projects, and then how to deploy the finished work. We will go over these core concepts so that students feel more comfortable writing code whether it's in Node.js or Pearl...

The class will go over:

What are programming languages
What are tech stacks
Pseudocode
Wire Framing
Waterfall vs Agile Methodology
UI vs UX vs Design
API's
